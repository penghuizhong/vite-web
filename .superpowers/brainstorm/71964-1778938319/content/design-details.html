数据流

用户操作 → 状态变化 → UI 响应

1. 点击气泡按钮 → setChatPanelOpen(true) → 面板展开
2. 发送消息 → chatStore.sendMessage() → 创建会话 → 侧边栏自动新增
3. 侧边栏点击会话 → setActiveConversation(id) + setChatPanelOpen(true) → 面板展开对应聊天
4. 点击关闭按钮 → setChatPanelOpen(false) → 面板收起为气泡
